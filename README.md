# Database Web Application

Full-stack web application built with Node.js, Express, Handlebars, and MySQL.

## Features
- CRUD operations for customers, orders, menu items, and order items
- Relational database design with JOIN queries
- Dynamic UI with dropdown relationships between tables

## Tech Stack
Node.js • Express • Handlebars • MySQL

## Running Locally

1. Clone the repository

git clone https://github.com/yourusername/projectname

2. Install dependencies

npm install

3. Configure database connection in `database/db-connector.js`

4. Start the server

node app.js

5. Open in browser

http://localhost:3000
